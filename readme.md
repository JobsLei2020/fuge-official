## 一、技术栈

| 技术 | 用途 |
|------|------|
| **HTML5** | 页面结构 |
| **CSS3 + 自定义变量** | 样式设计（品牌色 `#DFFF00` 统一管理） |
| **Bootstrap 5.3** | 响应式布局（PC / 平板 / 手机自适应） |
| **jQuery + Slick Carousel** | 产品画廊轮播交互 |
| **Font Awesome 6** | 图标库 |
| **Google Fonts (Poppins)** | 字体方案 |

**核心特点：**
- 单页应用（Single Page Application），所有内容在一个 HTML 文件中
- CDN 加载第三方库，无需安装任何后端环境
- 纯静态前端，无需数据库、无需服务器端语言

## 四、部署方式

### 方式一：本地直接打开（最简单）
直接用浏览器双击 `index.html` 即可打开查看。（某些浏览器可能需要本地 HTTP 服务才能加载 Slick 轮播）

### 方式二：Python 本地服务器
```bash
# 在 fuge-replica 目录下执行
python -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 方式三：部署到生产环境

| 部署目标 | 适合场景 | 操作要点 |
|----------|----------|----------|
| **Nginx / Apache** | 正式上线 | 将整个文件夹上传到服务器 web 目录即可 |
| **Vercel** | 快速部署 | 免费，直接拖拽文件夹部署，自动 HTTPS |
| **Netlify** | 快速部署 | 免费，支持 drag & drop 部署 |
| **GitHub Pages** | 免费托管 | 推送到 GitHub 仓库即可自动发布 |
| **阿里云 OSS / 腾讯云 COS** | 国内访问 | 对象存储 + CDN 加速，适合国内外客户 |
| **飞书云空间** | 内部展示 | 可直接上传 HTML 文件 |
| **Docker + Nginx** | 容器化部署 | 编写 Dockerfile，构建镜像运行 |

**国内外客户访问建议：**
- 国内：使用阿里云 OSS + CDN（便宜且快）
- 国外：可以使用 Cloudflare Pages 或 Vercel（自带全球 CDN）
- 最佳方案：阿里云/腾讯云国内节点 + Cloudflare 海外节点，实现全球加速

### 方式四：Docker 部署示例
```dockerfile
FROM nginx:alpine
COPY . /usr/share/nginx/html
EXPOSE 80
```
```bash
docker build -t fuge-site .
docker run -d -p 80:80 fuge-site
```
